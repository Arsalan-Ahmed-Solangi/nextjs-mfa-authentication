# Next.js MFA Authentication

A secure **Next.js** authentication system with **Multi-Factor Authentication (MFA)** using authenticator apps such as Google Authenticator and Microsoft Authenticator.

## Features

* User Login & Registration
* MFA (2FA) Authentication
* QR Code Generation
* TOTP Verification
* Protected Routes
* JWT Authentication

## Libraries

* speakeasy
* qrcode
* jsonwebtoken
* bcryptjs
* axios
* Material UI

## Environment Setup

Create a `.env.local` file:

```env
NEXT_PUBLIC_APP_NAME=NextJS MFA Authentication
NEXT_PUBLIC_BASE_URL=http://localhost:3000
NEXT_PUBLIC_API_URL=http://localhost:3000/api

JWT_SECRET=your_secret_key
```

The application configuration is managed through:

```text
src/constants/config/env.config.js
```

## Database

Users table:

```sql
users (
  id,
  name,
  email,
  password,
  mfa_enabled,
  status,
  created_at,
  updated_at
)
```

## Run Project

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## MFA Flow

1. Login with email & password.
2. Scan QR code using an authenticator app.
3. Verify OTP generated by the app.
4. Access protected resources securely.
